# LCDCONTRAST

Action

Set the contrast of a TEXT LCD.

Syntax

LCDCONTRAST x

Remarks

X | A variable or constant in the range from 0-3.  
---|---  
  
Some LCD text displays support changing the contrast. Noritake displays have this option for example.
