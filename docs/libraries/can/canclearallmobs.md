# CANCLEARALLMOBS

Action

Clear all Message Objects.

Syntax

CANCLEARALLMOBS

Remarks

Use CANCLEARALLMOBS after you reset the CAN controller to set all registers in the proper state. All registers belonging to the MOB will be clear (set to 0).
