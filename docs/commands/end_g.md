# End_G

Action

End drawing a graphics primitive.

Syntax

End_G

Remarks

It is recommended to have an End_G for each [Begin_G](begin_g.md). 

For advanced users you can avoid the usage of End_G in order to save extra graphics instructions in the

Display List RAM.
