# CmdColdStart

Action

This command sets co-processor engine to reset default states.

Syntax

CmdColdStart

Remarks

Example

' Pseudocode

CmdFgColor &H00C040

CmdGradColor &H000000

CmdButton 2, 32, 76, 56, 26, 0, "custom"

CmdColdStart

CmdButton 82, 32, 76, 56, 26, 0, "default"
